Como rodar esse trabalho:

- Crie uma pasta chamada Prova dentro do htdocs do XAMPP.
- Coloque todos esses arquivos dentro da pasta.
- Ligue o Apache no XAMPP.
- Acesse o navegador: http://localhost/Prova/
- Clique no botão Iniciar para abrir o sistema e testar.
